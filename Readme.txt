The Adobe Edge Animate 2.0.1 runtime is provided here for your reference.

In order to substitute this runtime for the minified equivalent you will 
need to edit the xyz_edgePreload.js file generated by Animate 2.0.1. You 
should replace edge.2.0.1.min.js with edge.2.0.1.js.

The file 'edge.preload.js' is provided for reference only. The code 
contained in this file is copied into the  xyz_preload.js file generated 
by Animate based on the contents of your project. The contents of 
yepnope.1.5.4 are also included in xyz_preload.js.

Deutsch
Die Adobe Edge Animate 2.0.1-Laufzeit steht hier zu Referenzzwecken zur Verfügung.

Um die verkleinerte Entsprechung durch diese Laufzeit zu ersetzen, müssen Sie die von Animate 2.0.1 generierte Datei „xyz_edgePreload.js“ bearbeiten. Ersetzen Sie „edge.2.0.1.min.js“ durch „edge.2.0.1.js“.

Die Datei „edge.preload.js“ dient nur zu Referenzzwecken. Der in dieser Datei enthaltene Code wird in die Datei „xyz_preload.js“ kopiert, die Animate entsprechend dem Inhalt Ihres Projekts generiert. Außerdem enthält „xyz_preload.js“ den Inhalt von „yepnope.1.5.4“.
